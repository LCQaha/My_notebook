# Windows 文件共享指南（新手版）

这是一份从 0 开始的教程。按步骤做，不需要懂网络术语，也能把电脑里的文件共享给同一局域网的其他设备访问。

适用系统：
- Windows 10
- Windows 11

目标：
- 在电脑 A 上创建一个共享文件夹
- 在电脑 B（或手机/平板）上访问它
- 保障基本安全，避免“谁都能删文件”

---

## 1. 先搞清楚：文件共享到底是啥

文件共享就是把你电脑里的某个文件夹，通过局域网“开放一个入口”给其他设备看/拷贝/修改。

你需要满足 3 个条件：
1. 两台设备在同一个网络（同一个路由器、同一个 Wi-Fi 或同一个交换机）。
2. Windows 开了“网络发现”和“文件和打印机共享”。
3. 共享权限和文件夹权限配置正确。

---

## 2. 准备工作（先做这个，别跳）

### 2.1 确认网络是“专用网络”

1. 打开 `设置` -> `网络和 Internet`。
2. 点击当前连接（Wi-Fi 或以太网）。
3. 把“网络配置文件类型”设为 `专用网络`（Private）。

说明：如果是“公用网络”，共享功能常常被系统限制。

### 2.2 开启网络发现和文件共享

1. 打开 `控制面板` -> `网络和共享中心`。
2. 左侧点 `更改高级共享设置`。
3. 在“专用”网络下，开启：
   - `启用网络发现`
   - `启用文件和打印机共享`
4. 在“所有网络”里，建议：
   - 临时测试可关闭“密码保护共享”
   - 长期使用建议开启“密码保护共享”（更安全）

---

## 3. 在电脑 A 上创建共享文件夹（核心步骤）

### 3.1 创建文件夹

1. 比如在 D 盘新建文件夹：`D:\Share`
2. 在里面放几个测试文件（txt、图片都行）

### 3.2 设置“共享权限”

1. 右键文件夹 `D:\Share` -> `属性`。
2. 打开 `共享` 选项卡 -> 点 `高级共享`。
3. 勾选 `共享此文件夹`。
4. 共享名可改成简单名称，比如：`Share`。
5. 点 `权限`，按需设置：
   - `Everyone` + `读取`：别人只能看/复制，不能改（推荐默认）
   - `Everyone` + `更改`：别人可新增、改名、删除
   - `完全控制`：权限最大，不建议随便开

### 3.3 设置“安全权限（NTFS）”

共享能不能访问，不只看“共享”页，还要看“安全”页。

1. 在 `属性` -> `安全` 选项卡里，查看用户权限。
2. 如果共享给 `Everyone` 读取，那这里也至少要给读取权限。
3. 规则记住一句话：  
   **最终权限 = 共享权限 和 安全权限 的交集（取更严格的）**

示例：
- 共享页给“可写”，但安全页只给“读取” -> 实际只能读取。

---

## 4. 在电脑 B 上访问共享文件

你可以用两种方式访问：

### 方式 A：直接输入地址

1. 在资源管理器地址栏输入：
   - `\\电脑A的IP\Share`
   - 或 `\\电脑A主机名\Share`
2. 回车访问。

示例：
- `\\192.168.1.20\Share`

### 方式 B：映射为网络驱动器（更方便）

1. 打开“此电脑” -> 顶部点 `...` 或 `计算机` -> `映射网络驱动器`。
2. 盘符选 `Z:`（随意）。
3. 文件夹填：`\\192.168.1.20\Share`
4. 勾选 `登录时重新连接`。
5. 完成后会像本地磁盘一样使用。

---

## 5. 怎么查看电脑 A 的 IP 和主机名

### 查 IP

1. `Win + R` 输入 `cmd`，回车。
2. 输入：`ipconfig`
3. 找到当前网卡下的 `IPv4 地址`，例如 `192.168.1.20`

### 查主机名

命令行输入：`hostname`

---

## 6. 手机/平板访问（可选）

如果手机支持 SMB（很多文件管理器支持）：
1. 在文件管理器里选“局域网/SMB/网络存储”。
2. 地址填：`192.168.1.20`
3. 共享名：`Share`
4. 输入 Windows 用户名和密码（若启用了密码保护共享）。

---

## 7. 最常见问题排查（照着对）

### 问题 1：看不到对方电脑

检查：
1. 两台设备是否同网段（如都在 `192.168.1.x`）。
2. 网络类型是否是“专用网络”。
3. 是否开启网络发现。
4. 重启一次两台电脑的网络连接（最简单：断开再连）。

### 问题 2：提示“无权限访问”

检查：
1. 共享权限是否有 `Everyone`（或你登录的用户）。
2. 安全（NTFS）权限是否也允许读取。
3. 是否开启了密码保护共享；如果开启，需要输入电脑 A 的账号密码。

### 问题 3：能打开但不能新建/删除

原因：一般是权限只给了读取。  
处理：在“共享权限”和“安全权限”里同时给“修改/写入”。

### 问题 4：突然访问失败

检查：
1. 电脑 A 是否休眠/关机。
2. IP 是否变化（重启路由器后常见）。
3. 防火墙是否拦截了文件共享。

---

## 8. 防火墙和安全建议（非常重要）

### 8.1 防火墙建议

- 保持 Windows 防火墙开启，不要直接全关。
- 确认“文件和打印机共享”在专用网络下被允许。

### 8.2 安全建议（长期使用）

1. 尽量在“专用网络”下共享，不在公共 Wi-Fi 下共享。
2. 优先开启“密码保护共享”。
3. 不要给 `Everyone` 完全控制，除非你非常确定环境安全。
4. 共享目录尽量单独建，不要直接共享整个系统盘。
5. 重要数据先备份，再开放写权限。

---

## 9. 进阶：只给指定账号访问（推荐）

比 `Everyone` 更安全的方式：

1. 在电脑 A 上创建本地用户（例如 `share_user`）。
2. 给该用户设置强密码。
3. 在共享权限和安全权限里，只给这个用户授权。
4. 电脑 B 访问时输入：`电脑A主机名\share_user` + 密码。

这样可以做到：谁访问了、权限多大、可追踪。

---

## 10. 不想共享了，如何关闭

1. 右键共享文件夹 -> `属性` -> `共享` -> `高级共享`。
2. 取消勾选 `共享此文件夹` -> 确定。
3. 如有映射驱动器，在客户端右键断开即可。

---

## 11. 一页速查（懒人版）

1. 设网络为专用。
2. 开网络发现 + 文件共享。
3. 新建 `D:\Share`。
4. 高级共享勾选共享，给读取权限。
5. 安全页也给对应读取权限。
6. 客户端访问 `\\IP\Share`。
7. 出问题先查：同网段、权限、防火墙、密码共享。

---

如果你要的是“家庭内多人可编辑”的方案，可在此基础上再加：
- 固定 IP（避免地址变动）
- 专用账号分权（读写分离）
- 定时备份（防误删）
