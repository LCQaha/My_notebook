在Windows局域网下实现不同机器之间的文件共享，主要可以通过以下步骤完成：

### 步骤1：配置共享文件夹

1. **选择文件夹**：

   * 在你的电脑上，找到想要共享的文件夹。
   * 右键点击该文件夹，选择 **“属性”**。

2. **设置共享**：

   * 在 **“共享”** 标签下，点击 **“高级共享”**。
   * 勾选 **“共享此文件夹”**，并为该文件夹设置一个共享名称（其他计算机会通过这个名称访问共享文件夹）。
   * 你还可以点击 **“权限”** 按钮来设置文件夹的访问权限。比如设置为 **“所有人”** 或 **“指定用户”**，并根据需要配置 **读取** 或 **读取/写入** 权限。

3. **启用文件和打印机共享**：

   * 在 **控制面板** > **网络和共享中心** > **更改高级共享设置** 中，确保 **“启用文件和打印机共享”** 选项已勾选。

### 步骤2：配置防火墙设置

1. 打开 **Windows 防火墙** 设置。
2. 确保允许文件和打印机共享通过防火墙：

   * 进入 **控制面板** > **Windows 防火墙** > **允许应用通过防火墙**。
   * 在应用列表中，确认 **“文件和打印机共享”** 已勾选。

### 步骤3：确保计算机处于同一网络

1. **检查网络连接**：

   * 确保两台计算机都连接到同一局域网内（通过Wi-Fi或局域网连接）。
2. **检查工作组设置**：

   * 确保两台计算机处于同一 **工作组**。
   * 在 **控制面板** > **系统** > **计算机名称** 中检查工作组名称，如果不一致，可以点击 **更改设置** 来修改工作组名称，使其一致。

### 步骤4：访问共享文件夹

1. **通过资源管理器访问**：

   * 打开资源管理器，在地址栏中输入 `\\计算机名` 或 `\\IP地址`（例如 `\\192.168.1.2`），然后按回车。
   * 你将看到该计算机上共享的文件夹。点击共享文件夹，即可访问。

2. **映射网络驱动器**：

   * 如果你希望更方便地访问共享文件夹，可以右键点击 **“此电脑”**（或 **“计算机”**），选择 **“映射网络驱动器”**。
   * 在映射驱动器窗口中，输入共享文件夹的网络路径（例如：`\\192.168.1.2\sharedfolder`），并选择一个驱动器字母。
   * 点击 **完成**，该共享文件夹就会出现在资源管理器中，作为一个可用的驱动器。

### 步骤5：使用用户名和密码（如需）

* 如果共享文件夹设置了权限，其他计算机在访问时可能需要输入计算机用户名和密码。
* 确保两个计算机上有相同的用户名和密码，或者在访问时输入相应的凭据。

通过上述步骤，你就可以在Windows局域网下实现不同机器的文件共享了。

2. **输入凭据**：

## 0. 你要实现的目标（对应配置思路）

你希望做到的是：

1. 在同一局域网（家里/办公室）里共享某个文件夹
2. **别人访问必须输入用户名/密码**（不能匿名）
3. 你能用“专用账户”控制别人能看哪些共享、能不能写
4. 以后想改密码/禁用某人访问，要 **简单、可控、可回收**

对应的实现方式是：
**Windows SMB 文件共享 + 启用“密码保护共享” + 为访问者创建本机账户 + 在共享权限与NTFS权限里只给这些账户授权**。

## 1. 局域网基础检查（先确保能互相找到）

在你的共享主机（A）和访问电脑（B）上都确认：

### 1.1 在同一网段

* A、B 都连同一个路由器/交换机（Wi-Fi/网线都行）
* A 上运行 `ipconfig` 看 IPv4，比如 `192.168.1.10`
* B 上运行 `ipconfig` 看 IPv4，比如 `192.168.1.20`
  只要前 3 段一致（如都 `192.168.1.*`），一般就是同网段

### 1.2 网络类型设为“专用网络”

* 设置 → 网络和 Internet →（Wi-Fi/以太网）→ 网络属性
* 选择 **专用网络（Private）**
  （公用网络会更严，容易访问不到）

## 2. 打开 Windows 的共享开关（关键：密码保护共享）

在共享主机 A：

1. 控制面板 → 网络和 Internet → 网络和共享中心
2. 左侧：**更改高级共享设置**
3. 在“专用(当前配置文件)”里：

   * ✅ 启用网络发现
   * ✅ 启用文件和打印机共享
4. 在“所有网络”里：

   * ✅ **启用密码保护的共享**（这一步就是“必须凭据”的核心）
   * （可选）关闭公用文件夹共享（避免无意开放）

## 3. 创建“专用访问账户”（给别人用的凭据）

你有两种常用策略：

* **策略A：一个“共享访问账户”给一类人用**（省事，但多人共用同一密码，不便追踪）
* **策略B：每个人一个账户**（最推荐：可单独改密码/禁用/审计）

### 3.1 在 A 上创建本地用户（Windows 10/11 通用）

**方法1（更直观）：设置界面**

* 设置 → 账户 → 家庭和其他用户（或“其他用户”）
* 添加账户 → “我没有此人的登录信息” → “添加没有 Microsoft 帐户的用户”
* 输入用户名、密码（建议强密码）

**方法2（专业版常用）：计算机管理**

* Win+X → 计算机管理
* 本地用户和组 → 用户 → 右键“新用户”
* 设置用户名/密码（建议不要勾“密码永不过期”，便于管理）

> 建议：账户只用于访问共享，不需要给管理员权限，也不需要能本地登录做事。

## 4. 共享文件夹：同时设置“共享权限”和“NTFS权限”（很多人只设了一半）

Windows 实际生效权限 = **共享权限** 和 **NTFS（安全）权限** 的“更严格者”。

### 4.1 打开高级共享

在 A 上选你要共享的文件夹（例如 `D:\Share`）：

* 右键文件夹 → 属性 → **共享** → **高级共享**
* ✅ 勾选“共享此文件夹”
* 设置共享名（例如 `Share`）

### 4.2 设置“共享权限”

* 高级共享 → 权限
* 建议做法：

  1. **删除** “Everyone/所有人”（避免匿名或误授权）
  2. 点击“添加” → 输入你创建的账户名（例如 `user1`、`guestshare`）→ 确定
  3. 给权限：

     * 只读：勾“读取”
     * 可写：勾“更改”(Change) + “读取”
     * 不建议给“完全控制”（除非你非常确定）

### 4.3 设置“NTFS 安全权限”（属性→安全）

* 回到文件夹属性 → **安全** → 编辑
* 同样：

  1. 添加对应用户（user1/user2…）
  2. 给予需要的权限：

     * 只读：读取和执行/列出文件夹内容/读取
     * 可写：额外给写入/修改（或直接选“修改”）
  3. 确认 **不要给不该给的人**（例如 Everyone）

> 经验建议：
>
> * “共享权限”你可以设置得稍微宽一点（比如给“更改”），
> * 具体到每个用户能干什么，用 NTFS 权限细分更好管。

* 确认 **文件和打印机共享** 在“专用网络”下允许

## 6. 访问方电脑如何连接（输入凭据 / 映射网络驱动器）

在访问电脑 B 上：

### 6.1 直接访问

进入后你可能看不到“Wi-Fi/以太网”字样，但一般会有：

## 方案 3：用 PowerShell 一条命令强制改成 Private（最稳、最不吃界面）

在 **电脑B**：

1. 开始菜单搜索 **PowerShell** → 右键 **以管理员身份运行**
2. 先看当前连接配置文件：

```powershell
Get-NetConnectionProfile
```

你会看到 `InterfaceIndex` 和 `NetworkCategory`（Public/Private/DomainAuthenticated）

3. 把它改成专用（把 X 换成你看到的 InterfaceIndex）：

```powershell
Set-NetConnectionProfile -InterfaceIndex X -NetworkCategory Private
```

4. 再确认一次：

```powershell
Get-NetConnectionProfile
```

> 如果 `Get-NetConnectionProfile` 什么都不显示，说明这台 B 当前可能根本没连上任何网络（例如网卡驱动/适配器被禁用）。这种情况你得先把网络连接恢复起来。
